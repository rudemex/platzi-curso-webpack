# platzi-curso-webpack
Curso de webpack por platzi

#### Install dependencies
```sh
npm install
```

#### Run Build
```sh
npm run build
```

#### Run local
```sh
npm run build:local
```

#### Run External
```sh
npm run build:external
```

#### Run compile css
```sh
npm run build:css
```

#### Run extract css
```sh
npm run build:extract:css
```

#### Run multiple entry points
```sh
npm run build:multi
```

#### Run dev server
```sh
npm run build:dev
```

#### Run babel
```sh
npm run build:babel
```

#### Run images loader
```sh
npm run build:images
```

#### Run fonts loader
```sh
npm run build:fonts
```

#### Run videos loader
```sh
npm run build:videos
```
