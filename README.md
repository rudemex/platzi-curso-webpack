# platzi-curso-webpack
Curso de webpack por platzi

Install dependencies
> npm install

Run Build
> npm run build

Run local
> npm run build:local

Run External
> npm run build:external

Run compile css
> npm run build:css

Run extract css
> npm run build:extract:css

Run multiple entry points
> npm run build:multi

Run dev server
> npm run build:dev

Run babel
> npm run build:babel

Run images loader
> npm run build:images

Run fonts loader
> npm run build:fonts

Run video loader
> npm run build:video
